What Missio Is

  • A VS Code Extension
  • A REST Client
  • Local & Git Native
  • Free & Open Source

What Missio Is Not

  • A Standalone IDE
  • A SaaS Platform
  • Cloud Synced
  • A Monthly Subscription

Built for developers

Everything you need to test APIs, nothing you don't.

File-Based Collections

YAML files you can read, diff, and version control. OpenCollection v1.0.0 compatible.

Environments & Variables

Switch between dev, staging, and prod. Variable highlighting, autocomplete, and click-to-inspect.

Secret Providers

Azure Key Vault integration with $secret syntax. No secrets in files, ever.

OAuth 2.0

Client credentials and password flows with automatic token management and live expiry countdown.

Visual Editors

Custom editors for requests, folders, and collections with native dirty indicators and undo/redo.

Import from Postman

Migrate your Postman v2.0/v2.1 collections and environments. Also import cURL, wget, and raw HTTP.
